# Am-editor example of Vue

This project use [Modern Vue template](https://github.com/byoungd/modern-vue-template)

## Usage

### Development

```
pnpm i

pnpm dev

```

### Build

To build the App, run

```bash

pnpm build

```

With Env:

```bash

pnpm build:test

```

And you will see the generated file in `dist` that ready to be served.
